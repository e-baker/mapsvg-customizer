The MapSVG Customizer adds a listing of posts within the popover element.

== Description ==

The MapSVG Customizer adds a listing of posts in a custom taxonomy within the popover element. It really is awesome.

== Installation ==

Installing "MapSVG Customizer" can be done by using the following steps:

1. Download the plugin
2. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
3. Activate the plugin through the 'Plugins' menu in WordPress
